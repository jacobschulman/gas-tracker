API Endpoints

  • GET /api/v1/latest.json — Current prices (national + all states)
  • GET /api/v1/national.json — Full national price history
  • GET /api/v1/states/{STATE_CODE}.json — History for one state (e.g. CA, TX)
  • GET /api/v1/fuel/{grade}.json — National history for one grade (regular, midGrade, premium, diesel, e85)
  • GET /api/v1/fuel/{grade}/states.json — All states history for one grade
  • GET /api/v1/fuel/{grade}/states/{STATE_CODE}.json — One grade, one state
Data sourced from AAA. Updated daily via GitHub Actions.
